# Automate IoTeX DePIN Workflows with Our New n8n Community Node

We're excited to announce the release of our latest n8n community node: n8n-nodes-IoTeX! As the decentralized physical infrastructure network (DePIN) space continues to grow, Velocity BPA is committed to making IoTeX automation accessible to everyone.

## Why We Built This Node

IoTeX is pioneering the future of DePIN and IoT devices on the blockchain, but integrating these powerful capabilities into automated workflows has been challenging. Teams need to connect IoTeX's W3bstream data processing, manage XRC tokens, and interact with Pebble trackers—all while maintaining complex workflow automations.

That's where our n8n-nodes-IoTeX comes in. We've bridged the gap between IoTeX's innovative DePIN ecosystem and n8n's powerful automation platform, enabling you to build sophisticated workflows without writing code.

## Key Features

Our IoTeX node brings comprehensive functionality to your n8n workflows:

- **W3bstream Integration**: Connect directly to IoTeX's decentralized data processing layer for real-time IoT device data
- **XRC Token Operations**: Manage and interact with XRC20 and other token standards on the IoTeX blockchain
- **Pebble Tracker Support**: Integrate data from Pebble trackers for location-based and environmental monitoring workflows
- **DePIN Device Management**: Automate interactions with IoT devices in the IoTeX ecosystem
- **Blockchain Interactions**: Query transactions, check balances, and trigger smart contract operations

## Real-World Use Cases

Imagine automatically processing data from distributed Pebble trackers, triggering notifications based on environmental conditions, managing token rewards for device operators, or building custom dashboards that pull real-time data from W3bstream—all without writing a single line of code.

## Getting Started

Installing the IoTeX node is simple. In your n8n instance, run:


npm install n8n-nodes-IoTeX


After installation, restart n8n, and you'll find the IoTeX node available in your node palette. Check out our GitHub repository for detailed documentation, examples, and contribution guidelines: https://github.com/Velocity-BPA/n8n-nodes-IoTeX
