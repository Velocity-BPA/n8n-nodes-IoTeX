**Version**: 1.0.0
**Author**: Velocity BPA

### Overview

A comprehensive n8n community node for the IoTeX blockchain, enabling workflow automation for DePIN (Decentralized Physical Infrastructure Network) applications. This package provides full support for IoT device management, W3bstream off-chain compute, native staking operations, and MachineFi machine economy operations.

### Key Features

- Complete IoTeX blockchain integration with 16 resource categories
- Native staking bucket management with Roll-DPoS delegate support
- W3bstream off-chain compute integration for ZK proofs and WASM applets
- Pebble Tracker hardware integration for IoT sensor data
- Cross-chain bridge support via Hermes/ioTube
- Real-time event monitoring through polling triggers
- Full address format support (io-address and 0x-address)

### Resources

- 16 resource categories
- 80+ operations
- 8 trigger event types

### Technical Stack

- TypeScript 5.3+
- n8n Community Node API v1
- ethers.js v6 for EVM compatibility
- axios for HTTP requests

### Target Users

- DePIN developers building IoT infrastructure
- n8n workflow automation specialists
- IoTeX ecosystem developers
- Enterprise IoT solution architects
- MachineFi application builders
